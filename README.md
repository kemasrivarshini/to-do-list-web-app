# To-Do List Web Application

This project is a simple To-Do List web application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Delete tasks
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the file `todo.html` in a web browser.

This project demonstrates basic front-end development and JavaScript DOM manipulation.
